# 키 입력 이벤트

37 : 왼쪽 화살표
39 : 오른쪽
38 : 위쪽
40 : 아래쪽


```javascript
$(document).keydown(function(e){
    switch (e.keyCode) {
        case 37:
            break;
        case 38:
            break;
        case 39:
            break;
        case 40:
            break;
    }
});
```
