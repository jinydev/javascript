# selectors API level1

querySelector
querySElectorAll



## querySelector
첫번째 요소 한개만의 선택할 경우

```js
var email = section.querySelector(".email");
```
클래스명이 email인 첫번째 요소를 선택합니다. 한개만 선택하여 반환을 하기 때문에 `[]` 배열 선택을 하지 않아도 됩니다.

## querySElectorAll
여러개의 요소들을 선택할 경우
반환값이 배열로 반환합니다.
